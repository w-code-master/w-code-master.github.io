Редакция от 16 марта 2026 года

Настоящее Пользовательское соглашение регулирует правила использования онлайн-сервиса Admirra.

Используя сайт https://admirra.ru и сервис Admirra, пользователь подтверждает, что ознакомился с настоящим соглашением и принимает его условия.

Если пользователь не согласен с условиями соглашения, он должен прекратить использование сервиса.

1. ОБЩИЕ ПОЛОЖЕНИЯ

Сервис Admirra представляет собой программное обеспечение, предназначенное для автоматизированного сбора, обработки и визуализации рекламной аналитики.

Сервис позволяет пользователям:

  • сбподключать рекламные аккаунтыора
  • получать аналитику рекламных кампаний
  • формировать отчёты
  • отправлять отчёты по email и Telegram
  • получать автоматические аналитические комментарии

Администратором сервиса является:

Индивидуальный предприниматель
Бурлаков Тимофей Геннадьевич

2. РЕГИСТРАЦИЯ В СЕРВИСЕ

Для использования сервиса пользователь должен создать аккаунт.

При регистрации пользователь обязуется:

  • указывать достоверные данные
  • не создавать аккаунты от имени третьих лиц
  • не передавать доступ к аккаунту другим лицам

Пользователь несёт ответственность за все действия, совершённые через его аккаунт.

3. АККАУНТ ПОЛЬЗОВАТЕЛЯ

Пользователь обязан обеспечивать безопасность своего аккаунта.

В случае подозрения на несанкционированный доступ пользователь должен немедленно уведомить администрацию сервиса.

Администрация сервиса вправе:

  • ограничить доступ к аккаунту
  • временно заблокировать аккаунт
  • удалить аккаунт

если обнаружены нарушения условий использования

4. ДОПУСТИМОЕ ИСПОЛЬЗОВАНИЕ СЕРВИСА

Пользователь обязуется использовать сервис только законным способом.

Запрещается:

  • использовать сервис для незаконной деятельности
  • пытаться взломать систему
  • обходить технические ограничения
  • использовать сервис для распространения вредоносного ПО
  • вмешиваться в работу сервера или инфраструктуры
5. ИНТЕГРАЦИИ С ВНЕШНИМИ СЕРВИСАМИ

Сервис Admirra может использовать интеграции со сторонними платформами.

К таким платформам могут относиться:

  • Яндекс Директ
  • VK Ads
  • системы аналитики
  • email-сервисы

Пользователь самостоятельно подключает такие интеграции.

Администрация сервиса не несёт ответственности за:

  • корректность данных сторонних платформ

В рамках отдельных тарифов сервис может предоставлять возможность использования режима White Label.

White Label означает использование сервиса под брендом пользователя.

Пользователь может предоставлять доступ к сервису своим клиентам.

При этом:

Сервис Admirra может устанавливать ограничения, включая:

  • количество подключённых аккаунтов

  • злоупотребление функционалом системы
