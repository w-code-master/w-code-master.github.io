Согласие даётся в соответствии с требованиями Федерального закона №152-ФЗ «О персональных данных».

2. ПЕРЕЧЕНЬ ПЕРСОНАЛЬНЫХ ДАННЫХ

Пользователь даёт согласие на обработку следующих персональных данных:

  • имя
  • адрес электронной почты
  • номер телефона
  • данные аккаунта пользователя
  • IP-адрес
  • данные cookies
  • технические данные устройства
  • данные браузера
  • данные об использовании сервиса
  • данные рекламных аккаунтов, подключенных пользователем
3. ЦЕЛИ ОБРАБОТКИ ПЕРСОНАЛЬНЫХ ДАННЫХ

Персональные данные обрабатываются в следующих целях:

  • регистрация пользователя в сервисе Admirra
  • предоставление доступа к функционалу сервиса
  • обработка платежей
  • отправка уведомлений и отчётов
  • техническая поддержка пользователей
  • улучшение качества работы сервиса
  • анализ использования сервиса
  • выполнение требований законодательства Российской Федерации
4. ДЕЙСТВИЯ С ПЕРСОНАЛЬНЫМИ ДАННЫМИ

Оператор имеет право совершать следующие действия с персональными данными:

  • сбор
  • запись
  • систематизация
  • накопление
  • хранение
  • уточнение (обновление, изменение)
  • использование
  • передача третьим лицам в рамках оказания услуг
  • обезличивание
  • удаление
  • уничтожение
5. ПЕРЕДАЧА ДАННЫХ ТРЕТЬИМ ЛИЦАМ

Пользователь соглашается с тем, что персональные данные могут передаваться третьим лицам, если это необходимо для функционирования сервиса Admirra.

Такими лицами могут быть:

  • платёжные системы
  • сервисы отправки электронной почты
  • сервисы аналитики
  • сервисы хостинга
  • рекламные платформы при подключении интеграций пользователем
  • сервис Telegram при отправке уведомлений

Передача данных осуществляется только в объёме, необходимом для оказания услуг.

6. СРОК ДЕЙСТВИЯ СОГЛАСИЯ

Согласие на обработку персональных данных действует с момента его предоставления и до момента его отзыва пользователем.

7. ОТЗЫВ СОГЛАСИЯ

Пользователь вправе отозвать согласие на обработку персональных данных.

Отзыв согласия может быть направлен оператору в письменной форме.

После получения отзыва оператор прекращает обработку персональных данных, за исключением случаев, предусмотренных законодательством.

8. ПОДТВЕРЖДЕНИЕ СОГЛАСИЯ

Пользователь подтверждает, что:

  • действует добровольно
  • предоставляет достоверные данные
  • ознакомился с Политикой конфиденциальности
